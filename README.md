# Guia Prático de CSS
Projeto desenvolvido durante as aulas da UC15 do Senac.

O objetivo do projeto é praticar os principais conceitos de CSS a partir de uma estrutura HTML pronta, aplicando estilos gradualmente e acompanhando os resultados diretamente no navegador.
