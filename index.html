Seletores e cores

Nesta aula utilizaremos seletores de tag, classe e ID para escolher elementos e aplicar cores.

Conteúdo desta aula

Seletor de tag

Este parágrafo pode ser selecionado pelo nome da tag.

Seletor de classe

Seletor de ID
